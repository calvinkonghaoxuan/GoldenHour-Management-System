# ⌚ GoldenHour Stock Operations Management System
## 🎓 WIX1002 Fundamentals of Programming Project (25/26)

> **"A robust Java-based solution for modern retail operations."**

### 📌 Project Overview
This project was developed for **GoldenHour**, a luxury watch retail company based in Kuala Lumpur. It facilitates daily store operations including inventory tracking, sales processing, and employee attendance management.

---

### ✨ Key Features

#### 🛡️ Authentication & Administration
* **Secure Login/Logout**: Role-based access for Managers and Employees.
* **Employee Registration**: Exclusive manager feature to register new staff with unique IDs.

#### ⏰ Attendance System
* **Clock-in/out**: Real-time attendance tracking.
* **Work Hour Calculation**: Automatically calculates daily total hours upon clock-out.

#### 📦 Inventory & Stock Control
* **Dual Stock Count**: Morning and night stock tallying with mismatch alerts.
* **Stock Movement**: Automated "Stock In" and "Stock Out" receipt generation (.txt).

#### 💰 Sales & Transaction
* **Sales Recording**: Captures customer details, items, and payment methods.
* **Auto-Update**: Real-time inventory deduction upon successful sales.
* **Digital Receipts**: Date-stamped sales records for easy auditing.

#### 🔍 Data Management
* **Advanced Search**: Search sales by customer name or stocks by model across all outlets.
* **Edit Records**: Flexibility to correct stock levels or sales information.

---

### 🚀 Advanced Features (Extra Marks)
* **Graphical User Interface (GUI)**: A fully functional desktop application interface for a seamless user experience.
* **Data Analytics**: Provides insights into best-selling models and daily revenue.
* **Persistent Storage**: All data is stored in localized text/CSV files, ensuring data is saved even after system termination.

---

### 📂 Project Structure
- **/backend**: Core business logic and system controllers.
- **/gui**: Java Swing/JavaFX view components and UI design.
- **/data**: Persistent storage files (CSV/TXT) for database simulation.

---

### 🛠️ Technologies Used
* **Language**: Java 17
* **Framework**: Java Swing / JavaFX (GUI)
* **Data Handling**: File I/O (CSV/TXT Processing)
* **Concepts**: OOP (Inheritance, Polymorphism, Encapsulation), Sorting & Searching Algorithms.
